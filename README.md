# 📝 To-Do List Web Application

A simple and interactive To-Do List application developed using HTML, CSS, and JavaScript. This project demonstrates dynamic DOM manipulation and basic event handling.

## 🚀 Features
- Add tasks dynamically
- Mark tasks as completed (strike-through)
- Delete tasks instantly
- User-friendly interface

## 🛠️ Tech Stack
- HTML (Structure)
- CSS (Styling)
- JavaScript (Logic)

## ⚙️ Working
- User enters a task in input field
- Task is added to the list using JavaScript
- Clicking a task marks it as completed
- Delete button removes the task from the list

## 📂 How to Use
1. Open `index.html` in any browser
2. Enter a task
3. Click "Add"
4. Click on task to mark complete
5. Click ❌ to delete

## 📌 Learning Outcomes
- DOM manipulation
- Event handling
- Dynamic element creation
